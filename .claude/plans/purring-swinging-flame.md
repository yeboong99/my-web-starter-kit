# 배포 아키텍처 문서 생성

## Context
로컬 개발 환경과 EC2 운영 환경의 배포 구조에 대해 논의한 내용을 루트 디렉토리에 MD 파일로 정리한다. 사용자가 다음 작업 후 다시 검토하고 구체적 계획을 세울 예정.

## 작업
- 루트 디렉토리에 `DEPLOYMENT_ARCHITECTURE.md` 파일 생성
- 대화에서 논의한 로컬/EC2 환경 구조, docker-compose 설정, Nginx, GitHub Actions CI/CD, Makefile, .env 전략 등을 포함

## 검증
- 파일이 루트 디렉토리에 정상 생성되었는지 확인
